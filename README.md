# Comparator Logic pe 4 biți (VHDL / FPGA)

Proiectarea și implementarea unui sistem logic în VHDL pentru compararea 
a două variabile pe 4 biți, cu testare hardware pe placa FPGA Basys 3.

## Tehnologii folosite
- VHDL
- Xilinx Vivado
- FPGA Basys 3

## Funcționalitate
- Compararea a două valori pe 4 biți (A și B)
- Semnale de ieșire: A > B, A = B, A < B
- Maparea intrărilor pe switch-uri
- Afișarea rezultatelor pe display multiplexat cu 7 segmente
